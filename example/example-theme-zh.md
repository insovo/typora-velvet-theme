# Velvet 主题示例（中文）

> 这份文档用于快速验证主题效果。请分别在 `Velvet Dark` / `Velvet Light` 下打开对比。

## 优势速览

- 结构扫描快：标题颜色分层明显
- 卡片一致性强：表格 / 引用 / 代码块视觉统一
- 长文更耐读：行高、留白、分割节奏更平衡
- 技术文档友好：行内代码与代码块辨识度高
- 文件树更顺手：侧栏接近 macOS Finder 风格

## 1. 标题与文本层级

这是一段正文，用于观察中文排版、标点、字重与段落节奏。
你可以检查 **加粗文本**、*斜体文本*、`行内代码` 与 ==高亮文本== 的可读性。

### 1.1 信息扫描能力

同样长度的内容里，层级色差越清晰，越容易快速定位重点。

- 技术笔记
- 学习文档
- 项目复盘

## 4. 代码（行内 + 代码块）

常见命令：`git status`、`pnpm build`、`npm run test`。

> 代码块右上角会以纯文字显示语言标签，简洁不遮挡。

```bash
# 快速查看主题变量
rg -n -e "--h[1-6]" -e "--sidebar" -e "--table-radius" -e "--font-" velvet-dark.css velvet-light.css
```

```json
{
  "theme": "velvet-light",
  "strengths": ["clear hierarchy", "card UI", "macOS-like sidebar"]
}
```

```css
:root {
  --table-radius: 12px;
  --sidebar-active-bg: #0a84ff26;
}
```

## 5. 表格（圆角卡片风）

| 维度 | Velvet Dark | Velvet Light |
|---|---|---|
| 阅读氛围 | 夜读沉浸 | 日读清爽 |
| 标题辨识 | 对比更强 | 层次更克制 |
| 引用/代码块 | 深色卡片更聚焦 | 浅色卡片更轻盈 |
| 导出文档 | 适合深色预览 | 更接近打印阅读 |
